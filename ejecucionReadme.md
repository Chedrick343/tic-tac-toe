#Ejecución de juego tictactoe.

Para ejecutar este juego es necesario ejecutar el siguiente comando: source pygame_env/bin/activate
Después de esto ejecutaremos el archivo gatoGUI con el comando: python3 gatoGUI.py
Se abrirá la interfaz gráfica y se mostrará el primer movimiento hecho por la IA.
El usuario puede colocar en las casillas su movimiento (fila, columna) siendo fila y columna mayor o igual que 0 y menor o igual a que 2.
Cuando el jugador elija su jugada presiona el botón colocar y será entonces el turno de la IA nuevamente.
